## Task Master 사용 가이드

이 PRD를 기반으로 태스크를 생성하려면:

1. 이 템플릿을 수정하여 실제 프로젝트 요구사항을 작성
2. 파일을 `.taskmaster/docs/prd.txt`로 저장 (또는 다른 이름)
3. 다음 명령어 실행:
   ```bash
   npx task-master parse-prd .taskmaster/docs/prd.txt
   ```

Task Master가 자동으로 이 PRD를 분석하여 구조화된 태스크 목록을 생성합니다.
